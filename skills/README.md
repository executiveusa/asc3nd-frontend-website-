## What these skills are

These are read-only reference files. They document standards that apply to this repo.

- **interactive-artifacts/SKILL.md** — UDEC 8.5 quality floor, sandbox constraints, Chart.js patterns, sendPrompt() handoff. Applies if building interactive widgets on the public site.
- **library/** — catalog of available agent skills relevant to this project.
- **review/** — Steve Krug usability audit lens for reviewing pages.
- **localization/** — Mexican Spanish conventions (tú vs. vos, MXN, culturally appropriate phrasing).
