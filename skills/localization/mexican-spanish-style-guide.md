# Mexican Spanish Style Guide

## For /es route content in this repo

## Core principles

- **Tú** (not vos, not usted) — conversational nonprofit tone
- **No Spain Spanish** — avoid "estáis", "me importa un bledo", etc.
- **Warm but direct** — nonprofit copy, not academic
- **Youth-facing language** — relatable, not patronizing
- **Mexican cultural awareness** — appropriate idioms if used at all

## Phrasing conventions

- Use gerunds for action: "impulsando", "construyendo" — not "para impulsar"
- Short sentences — mobile-first reading
- Avoid anglicisms where native Spanish exists:
  - "email" → "correo electrónico" (or just "correo")
  - "download" → "descargar"
  - "login" → "iniciar sesión"
  - "website" → "sitio web"
- Numbers: spell out one through nine; digits for 10+

## SEO / metadata

- Title: keep under 60 characters
- Description: keep under 160 characters
- H1: front-load the most important keyword
- Internal links: use descriptive anchor text in Spanish

## Accessibility

- All Spanish text must have equivalent ARIA labels if icons are used
- Form labels in Spanish, matching the input placeholder language
- Alt text on images: descriptive, in Spanish
