# Asc3nd Client Preview — Project Context

## What this is
A standalone static sales preview for Asc3nd Collective, a 501(c)(3) youth
development nonprofit (EIN 99-1881891) based in the Pacific Northwest. It is a
clickable demo of the public site vision — not the production Next.js app.

## Hard scope boundary
- Allowed to edit: `client-preview/` only.
- Off-limits: the Next.js production app, env files, secrets, package files,
  `--prod` deploys, `.vercel/.env*`, any folder outside `client-preview/`.

## Why static
A prior model run corrupted the Next workspace on Windows / WSL. The static
preview decouples the client-facing demo from that broken build path so the
client can see the vision now, while the real app is reconciled post-approval.

- Colors: black `#000000`, warm gold `#f5a617` (lighter `#ffb733`), white,
  cream `#fffaf2`, off-white `#f7f1e8`.
- Type: Barlow Condensed (display, 900 weight, uppercase) + Barlow (body).
- Wordmark: `ASC` + gold `3` + `ND` · `COLLECTIVE` · `ELEVATE. EMPOWER. ASC3ND.`
- Hero line: "Empowering youth. Elevating futures. Building community."
- Donate: https://www.givelively.org/
- Social (5): IG/FB/X/TikTok/LinkedIn signup URLs.
- Honesty rules: no fake stats, deadlines, partners, awards, checkout.
