# Asc3nd Client Preview

This folder is a standalone static sales demo for the Asc3nd public preview.

What it is:
- A polished HTML/CSS/JS preview
- Built for client-facing sales conversations
- Bilingual English / Mexican Spanish
- No backend, no checkout, no live Printify integration

How to view locally:
- Open `index.html` directly in a browser, or
- Serve the folder with any static host

Deploy target:
- Vercel static deployment from `client-preview/`

Notes:
- `llms.txt`, `robots.txt`, and `sitemap.xml` are included for preview discoverability
- This is a sales artifact, not the final production application
